## server rendering command

__windows)__

`...\server)` gradlew clean build

`...\server)` cd build/libs

`...\server\build\libs)` java -jar server-0.0.1-SNAPSHOT.war

__linux/macOS)__

`...\server)` ./gradlew clean build

`...\server)` cd build/libs

`...\server\build\libs)` java -jar ./server-0.0.1-SNAPSHOT.war


__url__: http://localhost:8080/index.html
